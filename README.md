# Newsly - Tech News App

A clean and simple web application that fetches the latest technology news. Built from scratch with a beginner-friendly structure. 

## Features Added (Milestone 4 Updates)
- **News Search**: Easily search through currently loaded news.
- **Sorting**: Sort news context from A - Z or Z - A.
- **Filtering**: View only the news articles you've 'liked'.
- **Responsive Layout**: Designed neatly with basic CSS Flexbox and Grid.
- **Dark Mode**: Simple dark mode switch that makes text easier to read at night.
- **Loading & Empty States**: Added basic indications when news is loading or no result is found.

## Tools and Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS, Fetch API, DOM manipulation)
- GNews API (for fetching top tech headlines)

## Running it Locally
1. Clone the repository.
2. Open `index.html` in your web browser.
3. Click "Get News" to fetch and display the data.
